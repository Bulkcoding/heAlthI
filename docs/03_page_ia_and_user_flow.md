- 운동 히스토리
- 다음 운동 추천
- 프로그램 분석

- 운동 / 프로그램 관리
- AI 추천 검수

## 2. 페이지별 목적

### 랜딩 페이지
- 서비스 가치를 짧게 전달
- 로그인 / 회원가입 전환 유도
- AI 추천, 운동 분석, 소셜 동기부여를 핵심 차별점으로 제시

### 로그인 / 회원가입
- 진입 장벽을 최소화
- 이메일, 소셜 로그인, 비밀번호 재설정 지원

### 대시보드
- 유저가 매일 처음 보는 메인 허브
- 오늘 운동, 최근 기록, AI 피드백, 주간 추세, 친구 활동을 한눈에 보여줌

### 오늘의 운동
- 오늘 해야 할 루틴의 전체 구성을 확인
- 예상 소요 시간, 운동 순서, 추천 세팅, 대체 운동 확인

### 운동 기록 시작
- 실제 기록 입력의 중심 화면
- 세트별 중량, 횟수, 성공 여부, RPE, 메모 입력
- 휴식 타이머와 이전 기록 비교 제공

### 운동 기록 상세
- 한 세션을 복기
- 총 볼륨, 수행 점수, PR, 개선점, 다음 세션 권장사항 확인

### 운동 히스토리
- 날짜 / 부위 / 프로그램 / 운동명 기준으로 과거 기록 탐색
- 장기 사용자의 재방문 핵심 페이지

### 다음 운동 추천
- 다음 세션 추천안을 확정하는 화면
- 추천 중량, 세트, 횟수, 루틴 조정 이유를 제공

### 프로그램 분석
- 웬들러, 스트롱리프트 등 방식별 분석
- 주차별 진행률, 정체 여부, 델로드 필요 여부 확인

### 친구 피드
- 친구 운동 완료, PR, 배지 획득 등 활동 확인
- 응원과 상호작용으로 리텐션 강화

### 내 프로필
- 누적 기록, 대표 성과, 배지, 공개 범위 관리

### 설정
- 계정, 알림, 프라이버시, 구독, 연결 계정 관리

### 장비 인식 업로드 / 결과
- 사진 업로드 후 기구 분류
- 연결 가능한 운동과 추천 루틴 진입

### 관리자 페이지
- 운영 안정성 확보
- 회원, 신고, 프로그램, AI 품질을 관리

## 3. MVP 사용자 흐름

### 신규 유저 흐름
1. 랜딩 유입
2. 회원가입
3. 온보딩 입력
4. 초기 프로그램 추천
5. 대시보드 진입
6. 첫 운동 시작
7. 기록 저장
8. 추천 확인

### 기존 유저 일상 흐름
1. 로그인
2. 대시보드에서 오늘 운동 확인
3. 운동 시작
4. 세트 기록
5. 운동 종료
6. 세션 요약 확인
7. 다음 운동 추천 수락 또는 수정
8. 필요 시 친구 피드 공유

### 분석 흐름
1. 대시보드 또는 히스토리 진입
2. 최근 세션 요약 확인
3. 프로그램 분석 진입
4. 주간 / 월간 성과 확인
5. 개선 액션 확인

### 장비 인식 흐름
1. 카메라 또는 이미지 업로드
2. 장비 인식 결과 확인
3. 장비 수정 또는 확정
4. 추천 운동 확인
5. 기록 시작 화면 이동

### 소셜 흐름
1. 친구 활동 피드 진입
2. PR / 운동 완료 이벤트 확인
3. 응원 또는 프로필 방문
4. 나의 기록 공유
